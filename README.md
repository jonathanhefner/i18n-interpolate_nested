# i18n-interpolate_nested

I18n interpolation support for nested values.


## Installation

Add this line to your application's Gemfile:

```ruby
gem "i18n-interpolate_nested"
```

And then execute:

```bash
$ bundle install
```


## Contributing

Run `rake test` to run the tests.
